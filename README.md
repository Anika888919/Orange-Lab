# 🍊 Orange-Lab

This repository contains screenshots and outputs from experiments performed using **Orange Data Mining**, a visual programming tool for machine learning and data analysis.

## 📌 Project Overview

Orange is an open-source data visualization and machine learning platform that enables users to build workflows through a drag-and-drop interface. This project demonstrates different stages of a machine learning workflow using Orange.

## 🔧 Tools Used

- Orange Data Mining
- Logistic Regression
- Test & Score Evaluation
- Feature Selection

## 🚀 Workflow

1. Load dataset into Orange.
2. Select relevant columns/features.
3. Apply the Logistic Regression algorithm.
4. Evaluate model performance using the Test & Score widget.
5. Export the results.
